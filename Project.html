we did project on concept

IOT based indistrial battery management system

we use Aurdino uno microcontroller for interface

voltage sensors are used to measure the voltage

current sensors are used to measure the current

gsm module is used to get the message of condition of battery
buzzer is used to alert in case of emergency

Summary: We use to maintain the condition of the battery in online in case absent of monitor we has to get the condition info of the battery

message which we get is of whether the temp is increase or decrease and the condition is good or bad

The components used for the project are
Aurdino uno
Batteries
Voltage sensor
current sensor
gsm mdule
buzzer
